Constant

Constant Redeclaration

1. JavaScript const can not be redeclared (same as let)

Constant Scope

1. JavaScript const has block scope (same as let)

Constant Array and Object behavior

Constant Array:

1. value of constant array can be changed

2. new value can be added in constant array

3. But constant array can't be reassigned

Constant Objects

1. property of constant object can be changed

2. new property can be added in constant object

3. But constant object can't be reassigned
