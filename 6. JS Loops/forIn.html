For-in iterates over keys (property names) - Objects

For-of iterates over values (iterables only) - Arrays, Strings, Maps, Sets

For-in works on object: ✅ display every properties and methods

for-in works on array too: ⚠️ but shouldn't be used because it gives indexes as string

for-of works on Arrays, Strings, Maps, Sets (gives you value directly)

for-of does not work on Object (objects are not iterable by default)
