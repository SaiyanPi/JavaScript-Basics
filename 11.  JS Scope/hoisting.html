1. VARIABLES DEFINED WITH VAR ARE HOISTED TO THE TOP OF THE BLOCK, AND INITIALIAZED WITH UNDEFINED.

2. VARIABLES DEFINED WITH LET AND CONST ARE HOISTED TO THE TOP OF THE BLOCK, BUT NOT INITIALIZED.

let: Using a let variable before it is declared will result in a ReferenceError.

const: Using a const variable before it is declared, is a syntax error, so the code will simply not run.

JavaScript Initializations are Not Hoisted
